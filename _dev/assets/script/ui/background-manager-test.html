東洋における近代個人主義の
最大の実践者と言われたガスト北村が、
ルソーとカントの著作を通じて
キリスト教の影響を受けたのは
周知の事実である。
しかしあくまでも、
北村の思想の根幹をなすのは
日々のアルバイト経験である。
キリスト教の思想は、アルバイト経験を
独自の倫理体系に結実させるための
触媒の役割を果たしたに過ぎない点は
注意を要する。
