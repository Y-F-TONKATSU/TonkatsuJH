誰も私に問わなければ、
私は知っている。
しかし誰か問うものに
説明しようとすると、

とんかつとは何か？

TONKATSU は
ビデオアーティスト/身体表現パフォーマーの
山田.F.とんかつをはじめ、
アーティストくずれの
失業者たちが寄り集まって創業した
Android アプリメーカーです。
斬新なビジュアルを打ち出した
アプリやゲームで
スマートフォンの世界に
新しい地平を開くことを
ミッションとしています。
Web サイト 'とんかつ騎士団の公式ホームページ' は
アプリの制作やそのための勉強から派生する
実験作品や思考の表現の場として
生まれました。

世界イデオロギーとしてのとんかつととんかつ騎士団の公式ホームページ

サイト名『とんかつ騎士団の公式ホームページ』は
長野県松本市の実在の広場の名前に
由来します。
現代の最も偉大な預言者
マハル師の寝言によれば、
そもそもこの広場の名称は、
チンズへネベの最も偉大な神・大マハートの
450年前の寝言に由来するとされており、
17次元宇宙秩序の実存に
限りなく近い表現を与える名前と言えます。
にもかかわらず、
近年、この広場は
「信州大学 生協前広場」という
誤った名称が流布されていました。
また、記憶のある方も多いと思われますが、
当時「両手をハサミにカニ歩き」や
「両手をついてカエル跳び」が、
あろうことか子供向けのテレビ番組で、
忍術として紹介されるという
痛ましい事件が発生しました。
我が国固有の文化である
忍者の品格を貶める虚偽の風説が
かくも堂々と罷り通ることに
心を痛めた山田.F.とんかつは、
いつの日にか真理が勝利をおさめることを祈念し、
真なる名『とんかつ騎士団の公式ホームページ』の看板を
制作しました。
以来、とんかつが
世界の最も有力なイデオロギーの一つに
数えられていることは周知の事実です。
ここでは TONKATSU のアプリや Web サイト、
また、数々のイメージやビデオについて、
制作者とその基調コンセプト、
そして、そのテクノロジーやアートを
紹介します。
