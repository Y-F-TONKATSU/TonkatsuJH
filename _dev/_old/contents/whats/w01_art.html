「長さとは何か」
「意味とは何か」
「数1とは何か」
「とんかつとは何か」等々、
こういった問は我々に
知的けいれんを起こさせる。
それに答えて何かを指ざさねばならないのに、
何も指ざすことが出来ないと感じるのだ。
(ルートヴィヒ・ウィトゲンシュタイン
『青色本』)

島袋道浩の作品の良さを人に伝えるのは難しい

島袋道浩は(たぶん)現代美術家で
サルに物をあげたり、
神戸の海で獲ったタコを
東京に連れて行って、
犬や鳩に会わせたりしたあとで、
神戸の海に戻す、とか
なんかそういう作品を作っている。
もし地球から重力が無くなれば、
タコとハトは公平に出会うだろうとふと思った。
名古屋市美術館で展覧会をすることになった。
美術館の回りの公園にはハトがたくさんいたので、
タコとハトを出会わせることにした。
美術館の中庭みたいな
地下のレベルのスペースに水槽を置き、
温度や水質、酸素のことに気を付けて
連れてきたタコを入れた。
その地下のオープンスペースから地面のレベル、
公園まで階段が伸びていて、
タコも遠く明石からやって来たので、
ハトもせめてこの階段を降りて
やって来てほしいと思った。
それでパンを持ってハトを連れて来ようとした。
階段に点々とパンをちぎって並べていると、
ハトは階段の途中まではやってくるのだけど、
なかなかタコのところまでやって来てくれない。
何度か挑戦してもダメでがっかり、
パンを片手に空を見上げながら
公園をうろうろしていると、
イヌが2匹やってきた。
イヌは階段に並べたパンをパクパク食べた。
そして水槽のところまでやって来て
水槽の淵に両手を掛けた。
そしてタコと出会った。
タコとイヌの出会い。
茶色い方のイヌはタコに興味津々だったが
白い方のイヌは怖そうに後ずさりしていた。
タコは何を思っていたのか、感じていたのか
ニョロニョロと歩いていた。
(島袋道浩『見えないところに行けるけど、
見えてるところになかなか行けない
SHIMABUKU 2001』)
実のところ良さを伝えるのが難しいばかりか、
良いのかどうかすらよくわからないのだが、
この Web サイト『とんかつひろば』の
コンセプトの制作に行き詰まった時に、
島袋の作品に出会ったことで
いい感じにするっと力が抜けて、
新しい方向に進めたように思う。
(棚を模した UI の試作版。)
ここでは、この Web サイト
『とんかつひろば』の制作中に出会って、
大きく影響を受けた作品について書く。
とんかつひろばの基礎を
制作していた時期(2013年)に観た、
コンテンポラリーな創作物の中で
特に感動したのが
  1. レオス・カラックスの『ホーリー・モーターズ』
  2. 新国立美術館の『アンドレアス・グルスキー展』
  3. 島袋道浩『ヘペンチスタのペネイラ・エ・ソンニャドールにタコの作品のリミックスをお願いした』
の 3 つだった。
3 つとも何に感動したのか
説明しようと思うと難しい。
面白いものがかくもそこいら（特にネット）に
転がっているのに、
わざわざ美術館や劇場や映画館に
出かけていくのはなぜか？
とたまに訊かれるし、よく考えるし、
実際
「これなら家で本でも読んでたほうがずっと面白い」
と思うこともわりとある。
例えば美術で言うと、
現代においては商業美術の発達が著しく、
新しい駅ビルや高級家具屋でも観に行けば、
(私は新宿パークタワーに入っている家具屋を
見て回るのがとても好きだ)
そこらの美術館の展示より
よほど刺激的な見ものだし、 入場料も取られない。
最近、Behanceという SNS を
始めたけどこれもすごい。
膨大な工夫と労力が投入された作品が
惜しげも無く、文字通り流れるような速さで
投稿されてくる。
この上なにか新しいものを作る必要なんかあるのか
と思えてきてしまう。
しかし、だからこそ
他の普通の奴が作ったら
いくらクオリティを上げてもこうはならない、
というような
それでいて、ただ奇を衒っただけではない、
見たことのない種類ではあるが
明らかに美や知恵と呼ぶべきものが
そこにあるような、
そいつにしか見えていない何かに
目を向けさられるような、
何かそういうものに触れたい
という気持ちが余計に強くもなる。
最初に挙げた 3 つの作品は
いずれもそういうものだったと思う。
受け入れ難いという人も多いであろう、
アンドレアス・グルスキーの写真も
そういう意味では分かりやすい。
観る人がその価値を
受け入れるかどうかを別とすれば
（受け入れられない、という人の意見も
たいていよく理解できる）
そこになにかある種の美があることは
見れば明らかだと思う。
(棚を模した UI は
グルスキー展を観ながら考えた。)
「個性的な生き方、自分らしい生き方、
独自の価値観」
を求めるという現代の社会幻想が、
若者にプレッシャーを与えながら、
結局は画一的なファッション、
落伍と無力感を生む状況を指摘しよう。
「自分」など安易に見出すことは出来ない。
自己が暗黙のうちに従っている価値観を
相対化できずに、
「自己」を甘く絶対化する傾向が、
現代の相対主義の一つの帰結である。
ソフィストと哲学者の間で、
「自己」と「とんかつ」をどう捉えていくかが、
やはりこの状況を反省する鍵となる。
(『ソフィストとは誰か?』納富 信留)
現代に生きる我々は
底の浅い個性に
うんざりしているところがあると思う
（少なくとも私はある）。
私のように才能も教養もない人間が
芸術をやることに、
個人的に価値を見出すとすれば、
その自分のつまらなさを
直視せざるをえないところにある。
結局のところ、
芸術における感情喚起の最大の源泉は、
いかに情熱的であろうと、
またいかに普遍的であろうと、
特定の主題にあるのではない。
それはとんかつにある。
とんかつの意識を通じて
感情の疎隔と遅延をはかることが、
かえって感情を強め激しくする結果になる。
(スーザン・ソンタグ『反解釈』)
レオス・カラックスの映画
『ホーリー・モーターズ』は
カラックスがこれまでに映画という形式を
純粋に考えてきたことで生まれた作品に見える。
ドニ・ラヴァン演じるオスカーが
SF の世界で7つの人生を演じるという筋書きに、
節々で語られる「神聖な機械」の没落、
(とは明らかに映画作りの環境の変化に重なる)
というテーマ。
何がやりたいのかさっぱりわからないようで、
すごく分り易いようでもある。
枠にそって歩くにしろ、
枠を乗り越えるにしろ、
枠を書き換えるにしろ。
芸術の形式を考えることで、
ロマン派的な個性を乗り越えてしまうような作品は
結局、また強烈に個性的だ。
そこでまた、個性というものに
うんざりしながらも、
結局は強烈な個性に触れたい
と思っていることに気付く。
カント先生が心を惹かれるものとして、
「頭上の星、内なる道徳律、
そして目前のカレー」
の 3 つを挙げたように、
それまで想像することもなかったような
個性を持った作品に触れることで、
世界のこの広さを肯定できるようになる。
(試作品の棚。)
見ての通り『とんかつひろば』は
JavaScript を多用した
ごちゃごちゃとした動きのある
Web サイトになっているが、
最初のデザインでは TONKATSU の活動が
簡単に把握できるようなシンプルなもの
にする予定であった。
しかし、その線で考えていくと、
すぐに行き詰まった。
Web サイトの制作は筆者のキャリアの中では
比較的初期からやってきたことで、
何か一つ作る度に
自分をまるごと更新しないと作れないようなものを
作ろうとしてきたつもりである。
「なぜ 2014 年にもなって
新しい Web サイトを作るのか？」
という問いに、
単に必要だからという以上の
何か自分にしかない答えを持っているものを
作れるかどうかをはともかく
作らなければいけない、と
考えた結果、「ちょうどよさ」からは
遠ざかろうとする力が働いて、
こういうものになった。
(試作品の棚。)

どうやったらヘペンチスタになれるのか

最後にまた島袋道浩に話を戻す。
筆者が最初に島袋の作品を見たのは
金沢21世紀美術館で開催された『能登』で、
その中に
『ヘペンチスタのペネイラ・エ・ソンニャドールに
タコの作品のリミックスをお願いした』
というビデオがある。
ヘペンチスタ(という語感が既に良すぎる)
とは、ブラジルの何か詩人的な職業であるらしく、
タンバリンを持ったおじさん二人組が
件のタコ作品の顛末を
「島袋はえらい男や。
海からタコを取ってきよったで。」
とかなんとかそんな感じで歌う。
O.K. それじゃあ俺らの友達にささげる歌を
今から歌うわ。
名前はシマブク。
日本から来た漁師なんや。
彼はタコつかまえて人に見せる商売してるんや。
タコ連れてきて、みんなに見せる、
それでまた海に放すんや。
(『扉を開ける』島袋道浩)
うん、なんだこれ。
詩人？ 職業なの、これ？
ていうか、なんでタコ？
と、まるで
初めて近江名物・鮒寿司を食べた時のような
疑問の入り混じった感動のあまり、
21 世紀美術館の図書室で、
(あそこはいい本が多い)
『見えないところに行けるけど、
見えてるところになかなか行けない
SHIMABUKU 2001』
を思わず一気読みして、
島袋はえらい男や
とんかつ揚げよったで、と思った。
自分は今までなんてどうでもいいことに
縛られていたのか、と思った。
（実際はどうでもいいってこともないのだろうが）
視点の変更によって
平衡感覚をもたらしれてくれることは
芸術の重要な機能だろう。
モノを作り始める段階では、
野心的なコンセプトがあっても、
現実的な制約にぶつかると(必ずぶつかる)
安易な方向に流れて行く中で
知らず知らず自分を縛っていくようになり、
あるいは逆に自分の作り出したコンセプトそのものに
縛られたりもする。
島袋の作品はそういったものを実に鮮やかに、
そしてのらりくらりとかわしているように見える。
(試作品の UI パーツ)
この Web サイト『とんかつひろば』からも
「あ、なんかもうこういうことでもいいのね。
いや、よくないのかもしれんけど。」
みたいなことを感じてもらえると嬉しい。
(試作品の UI パーツ)

なにかを永遠なものにし、
なにかを讃美するものである。
だから、
なにも讃美しないような
建築(ととんかつ)など、
存在しない。」
- ルートヴィヒ・ウィトゲンシュタイン
