とんかつメモ:Java の配列の宣言の書式

基本は
int[] numList = new int[3];
みたいな感じ。

中身も初期化するなら
int[] numList = new int[3]{1, 2, 3};
で、これは
int[] numList = {1, 2, 3};
こういう風にも書ける。

ただし宣言と初期化を切り離す場合は
new を省略できない。
int[] numList;
numList = new int[3]{1, 2, 3};
はいいが、
int[] numList;
numList = {1, 2, 3};
はダメ。
