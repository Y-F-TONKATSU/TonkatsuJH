とんかつメモ:JavaScript typeof

JavaScript では、
typeof xxx
で xxx の型を示す文字列を返す。
主な値は
"undefined", "boolean", "number",
"string", "object", "function" など。

typeof にはいくつか不便な点がある。
1 つは、
typeof null が "object" を
返す問題である。
null をチェックするには、
if(value === null)
という具合にする。

もう一つ、 配列に対しても "object" を返す。

new 演算子

JavaScript では コンストラクタは 基本的にただの関数であるが、 new 演算子を使って 呼び出した場合、 以下の通り 若干変わった挙動をする。
  • 空のオブジェクトが生成され、 this に代入される。 これが新しく生成される オブジェクトになる。
  • return がなくても 返り値が this, つまり新しく生成された オブジェクトになる。
