とんかつメモ:JavaScript Canvas

JavaScript Canvas の
width と heght の指定に関するメモ。
Canvas 要素との width/height プロパティと
CSS で設定できる width/height プロパティは
意味が違うので注意。
CSS での指定は、
Canvas の画素数を変えずに
要素の大きさを変える。
これはたいてい良くない結果をもたらすので、
基本 Canvas 要素の width/height プロパティを使う。
また、Canvas 要素の width/height プロパティは
値として整数値のみを受け付ける。
'px' などの単位を付ける必要はない。
width/height を指定しなかった場合、
300*150 の Canvas になる。
