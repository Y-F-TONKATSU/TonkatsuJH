Ticker

Adobe Animate で、 非常にシンプルなムービーを作り、 書き出してみる。

そんなわけで Ticker の中身は ただの requestAnimationFrame か setInterval である。 なので Ticker には、 Stage 等の CreateJS オブジェクト以外でも ループさせたい関数なら 何でもリスナーに登録していい。

デルタ時間を所得するには イベントオブジェクトの delta プロパティを参照する。 ↓こんな感じ。 

var sec = 0;

createjs.Ticker.addEventListener('tick', function(e){
	var current = Math.floor(e.delta * 0.001);
	if(current > sec){
		sec = current;
		console.log(sec + ' 秒経過!');
	}
});
